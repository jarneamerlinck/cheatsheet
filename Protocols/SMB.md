# Protocol SMB
[Readme](../README.md)
## Information

SMB = Server Message Block

- Network protocol 
  - TCP/IP
  - NetBIOS
  - NetBEUI
  - IPX/SPX
- Response-request protocol
- A client-server communication protocol 
- Sharing access to files, printers, serial ports and other resources
