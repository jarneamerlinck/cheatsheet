## Global flags

| What                                        | Switch   |
| :------------------------------------------ | :------- |
| get userlist                                | ```-U``` |
| get machine list                            | ```-M``` |
| get namelist dump (different from -U and-M) | ```-N``` |
| get sharelist                               | ```-S``` |
| get password policy information             | ```-P``` |
| get group and member list                   | ```-P``` |
| all of the above (full basic enumeration)   | ```-a``` |
